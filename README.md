# CSRes

C# Result type with related extension methods

## Concepts (goals)

- C# native (F# has it anyway)
  - pattern matching
- async/await ready
- collection extension methods
- Multiple values support (`(T1, T2)` and `(T1, T2, T3)` tuples) support
